CSS 布局技巧

2024-01-20 技术
css 前端

现代 CSS 提供了强大的布局工具：Flexbox 和 Grid。

Flexbox 布局

Flexbox 适合一维布局，可以轻松实现水平垂直居中。

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Grid 布局

Grid 适合二维布局，可以创建复杂的网格系统。

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}
