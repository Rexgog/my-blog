JavaScript 入门指南

2024-01-15 技术
javascript 编程

JavaScript 是世界上最流行的编程语言之一，也是 Web 开发的三剑客之一（HTML、CSS、JavaScript）。

基本语法

JavaScript 的变量声明使用 var、let 和 const 关键字。

let name = "博客";
const age = 25;

函数

函数是 JavaScript 的核心概念之一。

function greet(name) {
    return "你好，" + name;
}

DOM 操作

JavaScript 可以操作网页的 DOM 元素。

document.getElementById("title").textContent = "新标题";
