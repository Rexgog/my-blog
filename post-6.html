Git 使用笔记

2024-03-01 笔记
git 工具

Git 是现代开发不可或缺的版本控制工具。

基本操作

git init          # 初始化仓库
git add .         # 添加文件
git commit -m ""  # 提交

分支操作

git branch         # 查看分支
git checkout -b   # 创建并切换
git merge          # 合并分支

远程操作

git push origin main
git pull origin main
